Welcome to the GeoCroissant Generator documentation. This tool generates Croissant-compliant metadata with geospatial extensions for machine learning datasets.

## About

The GeoCroissant Generator extends the standard Croissant format with geospatial metadata, enabling better description and discovery of geospatially-enabled machine learning datasets.
